  • IBM MobileFirst
    Gathers a broad set of industry partnerships and resources for developers

    See the announcement

    Two new IBM MobileFirst Partner offerings:

    1. IBM Worklight Adapters for AT&T APIs
      • IBM Worklight developers can now create network-enriched mobile applications with AT&T APIs, including speech recognition (cross-carrier), SMS, notary management, in-app payment, and device capability. Plus they can code compelling native, HTML and hybrid mobile applications more quickly with more functionality.
      • IBM developerWorks community members can now access additional mobile and telecommunications content via the AT&T Developer Program.
    2. IBM technology will now be deployed to Nokia Siemens Networks (NSN) base stations

      This means the IBM WebSphere Application Service Platform for Networks (ASPN) now enables:

      • IT applications to run at the mobile network edge. Large data sets can be processed in realtime, which could be prohibitively expensive in a traditional cloud; and OTT services (over-the-top) are accelerated because of their proximity to user devices.
      • IT applications can be integrated with network interfaces. This means new network features can be rapidly developed outside a lengthy network infrastructure lifecycle; innovation is dramatically increased by fostering ecosystems of ISVs (independent software vendors); and applications can take advantage of rich radio network information not available today.
      • Smart cloud management. Standards-based cloud interfaces to manage applications can now extend to existing cloud processes at the edge of a network.
